# Visit manager

The goal of this app is to manage all the visits of the school from a client (the teacher) requirements.

## Installation

### Requirements

- Docker
- Docker-compose
- Git

### Steps

1. clone the project
```bash
git clone https://gitlab.isima.fr/bajacqueli/project-bases-de-donnees.git
```

2. Start the modules
```bash
cd project-bases-de-donnees
docker-compose up -d
```
